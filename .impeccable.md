# Impeccable Design Context

## Design Context

### Users
- **University Administrators**: Managing large datasets (students, courses, departments, attendance).
- **Professors (Doctors)**: Managing their lectures, attendance, and student interactions.
- **Context**: Daily use in a professional academic environment. Needs to be extremely simple, familiar, and distraction-free.

### Brand Personality
- **Ultra Minimal & Flat**: A very basic, clean theme. No aggression, no complex patterns.
- **Tone**: Calm, efficient, and standard. It feels like a "regular" control panel.
- **3-Word Personality**: Simple, Professional, Standard.

### Aesthetic Direction
- **Visual Tone**: Clean, flat, and light. High use of whitespace. Minimal use of borders.
- **Font Pair**: **Tajawal** (Display & Body). Clean and modern.
- **Theme**: Light-focused with a **Charcoal/Slate Dark Mode** (not jet black).
- **Differentiation**: Absolute simplicity. The interface stays out of the way, focusing entirely on data and tasks.

### Design Principles
1. **Absolute Simplicity**: If an element doesn't help the user complete a task, it doesn't belong.
2. **Soft Dark Mode**: Dark mode uses deep charcoal/slate tones (`oklch(0.2 0.01 250)`) for a comfortable, professional viewing experience.
3. **Standard Geometry**: Standard rounded corners (`0.5rem` / 8px) for a familiar, software-like feel.
4. **Subtle Contrast**: Rely on background tints and whitespace rather than heavy borders or shadows.
5. **Standard Typography**: No "shouting" with uppercase or black weights. Use standard casing and weights (`medium`, `semibold`) for a calm experience.
